# Эпилепсия: Современная Реабилитация

## Описание проекта

Комплексный информационно-образовательный портал по современным подходам к реабилитации при эпилепсии. Проект охватывает психотические и непсихотические формы, когнитивную реабилитацию и психологические интервенции для студентов-медиков и специалистов.

## Основные разделы

- Психотические формы эпилепсии
- Непсихотические формы
- Когнитивная реабилитация
- Психологические интервенции
- Неврологические аспекты
- Образовательные материалы (МКБ-11)

## Технологии

- VK Web App SDK
- VK UI Kit
- Responsive design
- Modern CSS Grid/Flexbox

## Автор

Проект разработан для образовательных целей в рамках программы по клинической психологии и психиатрии.
